Rin Sky

Rin Sky is a web application that allows you to track flights in real-time within a specified geographical zone. It utilizes the OpenSky Network API to fetch flight data and displays the flight positions on an interactive map. Whether you're an aviation enthusiast or not, Rin Sky provides a user-friendly interface to explore live flight information.
